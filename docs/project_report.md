## Overview

The AI Waiter is "context-aware". It doesn't just chat; it knows the **exact menu** available at the customer's table.

### Implementation

## ðŸ›’ Feature 2: Realtime Shared Cart
The cart is powered by **Supabase Realtime**. 
- **Session Based**: Users scanning the same table QR code join the same `cart_session`.
- **Live Sync**: Additions/removals update instantly across all devices at the table.
- **Persistence**: Cart state is saved in PostgreSQL, ensuring data isn't lost on refresh.
